The browser extension you have to get!

Features:

Style options:

  • Colour escalation 1
  • Colour escalation 2
  • Darkmode
  • Random font color
  • Kill images
  • Make website naked (disable css)

Socail media pages options (instagram/tiktok):

  • Download/View profile picture of this profile
  • Download/View picture of this post
  • Download/View video of this post
  • Download/View video of this insta story

Misc options:

  • Search marked text on several social media sites
  • Disable scripts in source code
  • Translate

Usage:

Just right-click in the browser window and click on "Seb's Extension".
Now click on the option you want to execute.
The most options can be stopped by clicking it again and if you reload the website the changes of the extension will be resetted.

Installation:

Hit the downlaod button below and save the .rar for example in your documents.
Unpack the rar.
Now enter in your browser "chrome://extensions".
Turn "Developer Mode" on (at the top right).
Click "load unpacked" and choose to the folder where you unpacked the rar.
Turn "Developer Mode" off (at the top right).
Make sure the extension is turned on and now it should be usable.
